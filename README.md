# 🤖 Gesture Controlled Robot

## 📌 Overview
This project demonstrates a gesture-controlled robot using an accelerometer and Arduino. The robot moves based on hand gestures, enabling wireless and intuitive control without physical interaction.

---

## 🚀 Features
- ✋ Gesture-based control system  
- 🔄 Real-time movement response  
- 📡 Wireless communication  
- 🤖 Forward, backward, left, right control  

---

## 🛠️ Tech Stack
- **Microcontroller:** Arduino Uno  
- **Sensor:** ADXL345 Accelerometer  
- **Motor Driver:** L298N  
- **Programming:** Arduino IDE (C++)  

---

## ⚙️ Components Used
- Arduino Uno  
- ADXL345 Accelerometer  
- L298N Motor Driver  
- DC Motors  
- Wheels & Chassis  
- Battery  

---

## 🔄 Working Principle
1. Accelerometer detects hand tilt direction  
2. Data is sent to Arduino  
3. Arduino processes movement direction  
4. Motor driver controls robot accordingly
